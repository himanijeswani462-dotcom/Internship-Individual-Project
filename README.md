Project Overview

This project focuses on exploratory data analysis (EDA) of the Online Shoppers Purchasing Intention dataset to understand customer behavior and factors influencing online purchase decisions.
The goal of this project is to practice real-world data analysis rather than building a highly optimized model.


Objectives
	•	Understand the structure and features of the dataset
	•	Analyze customer behavior related to purchasing
	•	Identify patterns and relationships using visualizations
	•	Apply EDA concepts in a practical project


Dataset

The dataset contains information about user sessions on an e-commerce website, including:
	•	Page visit counts and time spent
	•	Bounce and exit rates
	•	Visitor type
	•	Revenue (purchase outcome)


Exploratory Data Analysis (EDA)

The EDA includes:
	•	Data loading and basic cleaning
	•	Handling missing values
	•	Univariate and bivariate analysis
	•	Visualizations such as histograms, bar charts, box plots, and correlation heatmaps
	•	Key insights summarized using markdown cells
